**Metodologia - Geometria do Ser:**
Uma abordagem que compreende a vida como um sistema onde forças, cargas, tensões, energia e pontos de ruptura se integram.

**O trabalho investiga a engenharia interna que sustenta decisões, relações e direção** — e que, quando desalinhada, gera cansaço mesmo quando tudo "parece estar funcionando".

Desenvolveu uma leitura precisa dos desalinhamentos internos — **aqueles que não se resolvem com mais esforço, mas com reorganização de eixo**.
